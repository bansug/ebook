4장: Sigil 사용법

Sigil은 무료 오픈 소스 EPUB 편집기로, 윈도우, 맥 OS X, 리눅스 등 다양한 플랫폼에서 사용할 수 있습니다. Sigil을 사용하면 직접 HTML, CSS 코드를 작성하지 않고도 전자책을 만들 수 있습니다.

Sigil 설치하기

Sigil은 공식 웹사이트(https://sigil-ebook.com/sigil/)에서 다운로드할 수 있습니다. 운영체제에 맞는 버전을 다운로드하고 설치하세요.

Sigil 인터페이스

Sigil의 인터페이스는 크게 다음과 같은 부분으로 구성됩니다:

  • 북 브라우저: 전자책에 포함된 모든 파일을 볼 수 있습니다.
  • 편집 창: HTML 코드나 WYSIWYG 방식으로 콘텐츠를 편집할 수 있습니다.
  • 목차 편집기: 전자책의 목차를 편집할 수 있습니다.

Sigil로 전자책 만들기

다음은 Sigil로 전자책을 만드는 기본적인 단계입니다:

  1. 새 파일 만들기: File > New를 선택하여 새 EPUB 파일을 만듭니다.
  2. 메타데이터 편집: Tools > Metadata Editor를 선택하여 제목, 저자 등의 정보를 입력합니다.
  3. 표지 추가: Tools > Add Cover를 선택하여 표지 이미지를 추가합니다.
  4. 챕터 추가: File > New > HTML File을 선택하여 새 챕터를 만들고, 내용을 작성합니다.
  5. 스타일시트 편집: 북 브라우저에서 Styles 폴더를 찾아 스타일시트를 편집합니다.
  6. 목차 생성: Tools > Table Of Contents > Generate Table Of Contents를 선택하여 목차를 자동 생성합니다.
  7. 저장 및 검증: File > Save를 선택하여 저장하고, Tools > Validate EPUB를 선택하여 EPUB 파일이 유효한지 검증합니다.

Sigil의 고급 기능

Sigil은 다음과 같은 고급 기능도 제공합니다:

  • 코드 뷰: HTML, CSS 코드를 직접 편집할 수 있습니다.
  • 분할 뷰: 코드 뷰와 미리보기를 동시에 볼 수 있습니다.
  • 검색 및 바꾸기: 전자책 전체에서 텍스트를 검색하고 바꿀 수 있습니다.
  • 플러그인: 다양한 플러그인을 설치하여 기능을 확장할 수 있습니다.
